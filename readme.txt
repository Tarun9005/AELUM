0-Install Xampp.
1- The project is on PHP version 7.1.33
2- Make a database in localhost//phpmyadmin with name aelum.
3-Import the sql file in the database aelum to get the table.
4-Run xampp server.
5-Open in localhost to run the project.

6-It contains a registration form with timer of 3 minutes.
7-As soon as you click register the timer will start it's count.
8- Register yourself within 3 minutes.
9- See your entered details i.e name, email etc in the aelum database under userdetails table.
10-If you exceed 3 minutes then a message appears informing that time has passed and the registration form is no longer
displayed.
